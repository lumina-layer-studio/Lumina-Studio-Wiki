## About the team

We are a group of 3D-printing enthusiasts from across China. We originally came together to build a free and approachable tool for creating relief and print artwork. As the project developed, we realized that Lumina could also provide a useful workflow for the global 3D-printing community.

Whatever direction the project takes in the future, thank you for using Lumina Studio.
