## 开发组介绍
我们由一群分散在全国各地的3D打印爱好者组成，聚在一起的初衷是尝试做一款免费的可以让中国用户体验到更简单入门的版画/浮雕的生成软件，但随着开发的深入和不断的提升进步，我们明白这个项目或许可以进一步走进全球3D打印爱好者，为全球3D打印社区提供一个新的方案。
最后或许可以用一句特别的话结尾，无论未来Lumina项目命运如何，还是谢谢你们使用过我们的软件！
