# 内容许可与知识产权

除页面或素材另有说明外，Lumina Studio 对本 Wiki 中享有著作权的原创公开内容，
采用 [知识共享署名—非商业性使用—相同方式共享 4.0 国际许可协议
（CC BY-NC-SA 4.0）](https://creativecommons.org/licenses/by-nc-sa/4.0/) 许可。
你也可以查看 Creative Commons 提供的
[简体中文许可摘要](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.zh-hans)。

## 适用范围

本声明适用于本声明生效前已经发布、当前发布以及之后发布的原创公开 Wiki 内容，
包括 Lumina Studio 确实享有相应权利的原创文字、表格、示意图、标注和媒体。

页面、图片或视频如果另有署名、许可或“仅授权本 Wiki 使用”等说明，以该特别说明
为准。Lumina Studio 只能许可自己拥有或有权再许可的内容。

## 你可以做什么

在遵守许可条件的前提下，你可以：

- 复制并分享许可范围内的内容；
- 翻译、改写、重新排版或制作其他改编版本；
- 将内容用于学习、教学、社区交流和其他非商业场景。

## 使用时必须遵守

1. **署名**：注明 Lumina Studio Wiki，提供原页面和许可协议链接，并说明是否修改。
2. **非商业性使用**：不得将许可内容用于商业目的。
3. **相同方式共享**：发布改编版本时，继续采用 CC BY-NC-SA 4.0。
4. **不得增加限制**：不得使用额外法律条款或技术措施，阻止他人行使许可授予的权利。

推荐署名格式：

> 来源：Lumina Studio Wiki；原文链接：`<URL>`；许可：CC BY-NC-SA 4.0；
> 修改情况：`<说明修改内容或注明未修改>`。

## 不在许可范围内的内容

下列内容不因出现在 Wiki 中而自动采用 CC BY-NC-SA 4.0：

- Lumina Studio 或其他项目的源代码、构建与部署脚本、配置和基础设施；
- 第三方软件界面、Logo、商标、字体、模型设计、角色形象及品牌素材；
- 用户、测试者或合作方提供，并标注为仅授权本 Wiki 使用或采用单独许可的照片、
  视频和其他文件；
- 外部链接、嵌入视频以及引用的第三方文字和图片；
- 知识共享著作权许可未授予的专利权、商标权、隐私权、肖像权及其他权利。

上述内容继续受其各自条款及权利人的权利约束。引用或转载前，请查看页面附近的
署名和权利说明；没有明确许可时，不要把第三方素材当作开放内容使用。

## 软件代码与 Wiki 内容是两回事

CC BY-NC-SA 4.0 用于本 Wiki 的原创公开内容，不是 Lumina Studio 软件代码的许可。
复用软件代码、站点组件或第三方依赖前，请分别查看对应仓库适用条款和依赖许可。

## 投稿与素材提供

投稿者仍然拥有自己原创内容的著作权。向 Wiki 提交原创文字、表格、示意图或标注，
应确认有权允许其按本页面的许可发布。对于照片、视频、模型或其他素材，可以明确
选择 CC BY-NC-SA 4.0、其他许可，或仅授权 Lumina Studio Wiki 使用；发布页面应如实
记录署名和范围。

如发现署名遗漏、权利边界不清或疑似未经授权的内容，请通过 Wiki 公示的反馈渠道
联系 Lumina Studio，并附上页面链接和相关权利说明。
