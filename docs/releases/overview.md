# Lumina Studio 更新日志

这里记录 Lumina Studio 的公开版本历史，以及 2.0 内测阶段已经完成证据核对的开发进展。

:::info 当前状态
Lumina Studio 2.0 正在内部开发和分批验证，暂未确定最终功能范围和发布日期。“正在开发”不等于已经发布；月报只收录已经合并并经过整理的工作。
:::

## 最新进展

### [2026 年 7 月：云端资源、原生 SVG 浮雕与预览完善](/zh/docs/更新日志/2026-07)

7 月月报持续更新中，当前统计至 7 月 17 日。新增原生 SVG 浮雕、任务完成通知和图片旋转，继续完善云端 LUT、材料预览、矢量导出与 Snapmaker U1 兼容性。

## 1.x 公开版本

[查看 1.0—1.6.4 完整版本历史 →](/zh/docs/更新日志/1.x-history)

1.x 从 2026 年 1 月首次公开发布开始，先后完成统一应用、实时 3D 预览、多色 LUT、8 色/6 色校准、矢量建模、自由配色、浮雕与切片器集成。进入 2.0 内测开发后，后续进展改用月报记录。
