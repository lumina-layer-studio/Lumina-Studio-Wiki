# Lumina 2.0 怎样选择和保管耗材？

如果只记住一句话：**先选择机器能够稳定打印、身份能够确认、档案能够对应的真实线材，再考虑它在
屏幕上是不是最接近目标颜色。**

对 Lumina Studio 来说，“红色 PLA”还不够完整。材质、品牌、系列、商品颜色、批次、表面效果、
透光表现、当前含水状态和所用耗材档案，都会影响最终叠色结果。

## 买料或选料时，按五层顺序筛选

### 1. 机器、送料系统和喷嘴是否支持

先看线材制造商和打印机制造商的正式要求：

- 打印机和送料系统是否支持这种材料、线盘和软硬度；
- 喷嘴材质与直径是否适合其中的填料；
- 切片器里是否有适合当前机器的材料预设；
- 打印板、温度范围、封闭环境和通风条件是否满足要求。

发光、碳纤维、玻纤、金属粉、木粉和其他高填料线材，可能具有磨蚀性或更容易堵塞。不要因为
Lumina 能选择一个颜色，就默认 0.2 mm 喷嘴、普通黄铜喷嘴或自动送料系统也能安全使用它。
以该产品与设备说明为准，必要时先更换合适硬件。

### 2. 先确定材料类型，再选颜色

PLA 和 PETG 都可以成为彩色平面作品的材料，但不能只按“哪个颜色更像”来替换。两者的打印温度、
首层、流动、拉丝、表面和叠层表现可能不同。

同一作品中，**在没有验证跨材料组合前，优先使用同一材料体系**。如果确实需要混合 PLA、PETG
或其他材料，至少先确认：

- 打印机和切片器支持这组材料；
- 每个项目耗材保留了正确的材料预设；
- 相邻材料能否在当前结构中可靠结合；
- 清洗塔、换料温度和残料不会破坏后续颜色；
- 已经用小样验证，而不是直接打印正式作品。

有些设备文档会把 PLA 与 PETG 的低粘附用于支撑界面，这不等于它们适合在任何彩色主体中随意
叠加。Lumina 能生成项目耗材，也不构成对机器兼容、层间结合或安全性的保证。

### 3. 把表面效果和透光表现当作线材身份的一部分

同样写着“白色”或“红色”，下面这些线材不能默认共用一份档案：

| 类型 | 对 Lumina 结果的影响 | 使用前重点 |
| --- | --- | --- |
| 基础色 | 适合作为稳定对照，但不同品牌和批次仍会变化 | 核对品牌、系列、颜色和批次 |
| 哑光 | 添加剂和表面散射可能改变亮度与透光 | 单独验证，不沿用同色基础 PLA 档案 |
| 丝绸、金属光泽 | 观察角度和光线会明显改变外观 | 小样应在稳定光线下比较 |
| 透明、半透明、自然色 | 底板和下层颜色更容易透出 | 优先自制档案，并同时检查白底与黑底结果 |
| 发光、木粉、金属粉、碳纤维等填料 | 既影响光学表现，也可能增加磨损、堵塞或送料风险 | 先看硬件要求，再决定是否适合细喷嘴和多材料系统 |
| 再生料、混色料或批次波动明显的线材 | 同名线盘之间也可能出现差异 | 新批次先做小样，重要项目重新验证 |

### 4. 让真实线盘与耗材档案一一对应

选择耗材档案时，至少核对：

1. 材质；
2. 品牌和产品系列；
3. 商品颜色、色号和表面效果；
4. 档案测量时使用的层高；
5. 当前实际装入的线盘与槽位。

屏幕色块和 HEX 适合帮助人辨认，但它们只描述显示颜色，不能描述线材在不同厚度、底色和光线下的
真实表现。如何在现成档案、自制档案、反推 LUT 与传统 LUT 之间选择，请看
[现成耗材档案、自制档案、反推 LUT 与传统 LUT 怎么选？](pathname:///zh/docs/knowledge/material-profile-selection/)。

### 5. 最后用真实小样定稿

正式作品前，至少完成一次“小图 → Lumina 预览 → 切片器逐层预览 → 实物”的闭环。小样应该使用
正式作品相同的线盘、材料预设、喷嘴、显色层高和底板条件。

如果小样与预览明显不符，先检查线盘、档案、材料映射和受潮状态，不要立刻通过改原图来掩盖材料问题。

## TD 值与 Lumina 耗材档案是什么关系？

TD 是 Transmission Distance（透光距离）的缩写。在常见的耗材绘画工作流中，它用一个以毫米为
单位的值描述：实心线材需要达到多大厚度，才能阻挡绝大部分光线。一般来说，较低 TD 更快遮住下层，
较高 TD 允许下层在更多层内继续参与混色。

但 **TD 不是一卷料的完整颜色档案**：

- 它不能单独表达完整色相、亮度和三个颜色通道的差异；
- 它不能替代白底与黑底上的实际梯度；
- 它不能说明哑光、丝绸、填料和表面散射的全部影响；
- 测量方法、照明、厚度和批次不同，得到的值也可能不同。

Lumina 2.0 的耗材档案来自白底和黑底梯度卡的实际提取，并记录供当前预测流程使用的测量结果。
因此，网上找到一个 TD 值可以帮助理解线材透明程度，但不能直接替代与你手上真实线材对应的
Lumina 耗材档案。

### 开封时建立身份卡

建议在线盘或密封袋上记录：

- 材质、品牌、系列、颜色、色号和批次；
- 开封日期；
- 对应的 Lumina 耗材档案名称；
- 档案测试层高和最后验证日期；
- 干燥日期、异常表现与剩余量。

不要把几段剩余线材放进另一卷的包装袋，也不要只靠“这个蓝更深一点”区分。真实线盘一旦混淆，
再准确的软件档案也无法恢复对应关系。

### 密封保存不是主动干燥

不用时，把线材放进密封袋或密封箱，并配合仍然有效的干燥剂；减少不必要的开盖，并定期检查密封和
干燥剂状态。

干燥箱或密封箱主要用于**减缓再次吸湿、维持低湿环境**。已经受潮的线材通常需要主动干燥；只把它
放进装有干燥剂的箱子里，不代表水分已经从线材内部移除。

### 烘干温度不能使用一张通用表

不同材质、配方和线盘的耐温不同。温度过低可能没有效果，温度过高可能让线材粘连、线盘变形或产生
其他风险。因此：

1. 优先使用该线材制造商给出的温度与时间；
2. 同时确认线盘本身能承受该温度；
3. 使用能够稳定控温并适合耗材的设备；
4. 干燥后立即回到密封、低湿环境；
5. 把本次干燥记录到线盘身份卡。

### 拉丝不一定等于受潮

受潮可能伴随表面变差、气泡、爆裂声、料滴、层间结合变差或拉丝增加，但拉丝也可能来自温度、回抽、
空驶路径、材料本身或切片设置。

判断时一次只改一个条件。可以先用同一小模型对比“干燥前后”，再决定是否修改温度和回抽；否则容易
把多个原因混在一起。

## 打印前的一分钟检查

- 线盘标签与 Lumina 中选择的档案一致；
- 切片器项目耗材保留了正确材料类型；
- 真实槽位中装的是记录的那卷料；
- 线材没有明显受潮、脆断、打结或送料异常；
- 填料线材符合当前喷嘴与送料系统要求；
- 修改档案、层高或线盘后，已经重新预览和切片；
- 正式大图前已经完成小样。

生成后的对象与真实料槽如何对应，请继续阅读
[把 Lumina 3MF 正确导入切片器](pathname:///zh/docs/tutorials/3mf-slicer-material-mapping/)。

## 常见问题 Q&A

### PLA 和 PETG 都能用吗？

只要你的打印机、切片器和当前工作流支持对应材料，两者都可以分别建立和使用耗材档案。但同一档案
不能同时代表 PLA 与 PETG，同一作品跨材料混用也需要单独验证温度、送料、结合和切片设置。

### 没有完全同色的耗材，应该追一个最接近的 HEX 吗？

不用强行追求相同 HEX。先选机器能稳定打印、身份清楚并能取得可靠档案的线材，再用 Lumina 预览和
真实小样判断。显示色相近不代表透光和叠层结果相近。

### 同品牌同颜色，换了新批次还要重新校准吗？

不一定每卷都必须重做。先用已有档案打印小样；如果颜色、透光、表面或打印状态明显变化，重要作品
再为新批次重新验证或制作档案。

### 哑光 PLA 可以直接使用基础 PLA 的同色档案吗？

不建议默认共用。哑光添加剂会改变表面和光线散射，同色名称不能证明叠层表现相同。可以先试打，
对颜色一致性要求高时单独制作档案。

### 透明线材是不是 TD 越高越好？

没有“越高越好”。较高与较低的透光会产生不同混色范围和遮盖能力，应该根据作品、层高和其他线材
组合选择，并用档案和小样验证。

### 0.2 mm 喷嘴能不能打印木粉、碳纤维或发光耗材？

不要做统一假设。填料颗粒、磨蚀性和送料要求因产品而异，不少产品会要求硬化喷嘴或更大直径。
只有线材与打印机厂商明确支持当前喷嘴和送料系统时才使用。

### 干燥箱里有干燥剂，为什么线材还是拉丝？

密封和干燥剂主要减缓继续吸湿，不一定能把已经进入线材内部的水分主动排出。先按产品说明正确干燥，
再做对照测试；同时保留温度、回抽和路径等其他可能原因。
