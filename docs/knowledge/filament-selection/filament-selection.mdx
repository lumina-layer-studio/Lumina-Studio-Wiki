# 耗材使用
**Lumina目前支持2/4/6/8四种模式，现阶段是以固定搭配的形式进行使用，当然如果你理解色卡和校准的原理可以自行替换对应颜色。本篇就不过多介绍这种扩展玩法了。

:::warning 必看

这里说的颜色只需要接近，不是强行要求找到 HEX 值完全相同的耗材。例如，绿色可以选深绿或浅绿，红色可以选深红或普通红，蓝色可以选深蓝或浅蓝，黄色可以选淡黄或荧光黄，白色可以选奶白或普通白，青色也可以用青色、蓝色或薄荷色等接近颜色。

关键是在校准和使用时，始终保持软件中的颜色对象与实际耗材一一对应。

:::
