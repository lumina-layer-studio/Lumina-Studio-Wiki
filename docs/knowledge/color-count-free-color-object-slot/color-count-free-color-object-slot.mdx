# 颜色数、自由色、对象与料槽有什么区别？

如果只记住一句话：**这些数字描述的是不同层级，不需要相等。** 一张图片可以被量化成 48 个颜色组，
再用 4 卷真实耗材和 5 层叠色配方表现；生成的 3MF 又可能包含多于 4 个对象，最后映射到机器上
任意编号的 4 个实际料槽。

因此，不要用“页面选了多少颜色”直接推断“切片器里应该有多少对象”，也不要用项目耗材编号推断
它必须放进同号 AMS 槽位。

## 先用一张表分清八个概念

| 名称 | 它回答的问题 | 不代表什么 |
| --- | --- | --- |
| 原图颜色 | 图片文件里每个像素原本是什么 RGB | 不代表打印机能直接打印这些颜色 |
| 量化颜色数 | 高保真模式把相近像素归成多少个目标色组 | 不代表真实线盘数、AMS 槽位数或 3MF 对象数 |
| 匹配色 / 叠色配方 | 每个目标色最终用哪种可达颜色和层序表现 | 不代表必须为每个结果色准备一卷线材 |
| 已选耗材 | 当前 LUT 或耗材档案实际允许使用哪些材料通道 | 不代表它们必须占用同号物理槽位 |
| 叠色层数 | 一个像素配方沿 Z 方向最多使用多少个显色层 | 不代表选择了多少卷耗材 |
| 自由色 | 把某个已匹配颜色的区域额外拆为独立 3MF 对象 | 不会自动装料，也不等于无限增加可用实物颜色 |
| 3MF 对象 / 零件 | 切片器里可独立选择和分配耗材的几何结构 | 不等于项目耗材数；多个对象可以共用一份耗材 |
| 项目耗材与实际料槽 | 3MF 内部的材料定义，以及真实线盘所在的机器位置 | 两套编号不要求一致，必须在切片前确认映射 |

## 量化颜色数：先把连续图片整理成可处理的色组

照片、插画和抗锯齿边缘通常包含大量略有差异的 RGB。高保真模式不会为每个像素值分别求解，而是先用
颜色聚类把相近像素归成有限数量的目标色组，这就是“量化颜色数”。

当前转换器中，这个控件：

- 只对高保真模式的颜色量化生效；矢量模式不进行这一步；
- 默认可在 8–256 之间选择，也可以使用“自动计算”；
- 解锁后最高可到 8192，但会显著增加内存和处理时间，只适合确有必要的极端素材；
- 数值更高通常能保留更多渐变和低频细节，数值更低会形成更少的色块并加快处理。

量化颜色数不是“打印颜色数”。设为 48 的意思是软件最多把当前图像整理为约 48 个目标色组，**不是让你
准备 48 卷料**。后续多个目标色可以由同一组真实耗材使用不同叠层顺序表现。

:::tip[什么时候调整它]
如果大面积渐变被压成明显色带，可以在其他条件不变时适量提高；如果预览很慢、素材本身颜色简单，
可以降低。不要只因为原图分辨率高就直接解锁到 8192。
:::

## 匹配色与叠色配方：目标颜色不等于一卷实物线材

量化后，每个目标色还要映射到当前 LUT 或耗材档案能够达到的颜色。Lumina 可以用多层真实耗材的
顺序、重复和底色，让少量线材组合出更多视觉颜色。

例如，一张图有 48 个量化色，当前选择了白、红、黄、蓝 4 份耗材档案。不同目标色可能分别使用：

- 白 → 红 → 白 → 白 → 白；
- 白 → 黄 → 红 → 白 → 白；
- 白 → 蓝 → 红 → 黄 → 白。

这三种层序使用的仍然是同一组真实耗材，却会产生不同的最终颜色。因此：

- “画面上能看到多少种结果色”可以大于真实线盘数；
- “量化色数量”可以大于已选耗材数量；
- 多个量化色也可能匹配到相同或非常接近的可达色。

如果整体色域不足，应该先检查耗材档案和真实线材，而不是只增加量化颜色数。如何选择颜色来源，请看
[现成耗材档案、自制档案、反推 LUT 与传统 LUT 怎么选？](pathname:///zh/docs/knowledge/material-profile-selection/)。

## 已选耗材与叠色层数：一个决定材料通道，一个决定 Z 向配方长度

在耗材档案模式中：

- “已选耗材”是本次允许参与求解的真实材料档案；
- “底板耗材”指定用于底板或基底的材料身份；
- “叠色层数”是一个颜色配方沿 Z 方向最多包含多少层；
- “层高”决定每一层的实际厚度。

因此，选了 4 份耗材档案、使用 5 层叠色是完全正常的。5 层可以重复使用同一份耗材，并不要求准备
第 5 卷材料。

旧 LUT 名称里的 2 色、4 色、5 色、6 色或 8 色，主要描述这份 LUT 的材料通道和兼容方式，也不是
图片最多只能出现对应数量的视觉颜色。旧模式不能与下面三个数字混为一谈：

1. 高保真模式的量化颜色数；
2. 每个配方的叠色层数；
3. 机器当前可用的物理料槽数。

如果真实机器的可用料槽少于项目需要的材料通道，仍需根据设备能力决定是否换料、分盘或减少材料，
软件中的数字不会替硬件增加槽位。

## 自由色：把一个结果色区域拆成独立对象

“自由色”不是另一种量化模式。当前流程中，先完成预览并选中一个已经匹配的颜色，再把它标记为
自由色；重新生成时，Lumina 会把命中的区域额外生成一个名为 `Free_颜色HEX` 的独立对象。

它适合下面这类需求：

- 某个 Logo、文字或纯色色块必须使用一卷指定实物线材；
- 希望在切片器中单独选中一块区域，再手动分配项目耗材；
- 需要把特殊材料与普通叠色区域分开检查。

但标记自由色不会自动完成这些事情：

- 不会把量化颜色数加一；
- 不会自动创建或占用某个真实 AMS 槽位；
- 不会根据屏幕 HEX 自动找到你手上的同色线盘；
- 不会绕过打印机、材料和物理槽位限制。

标记或清除自由色后，旧的 3MF 下载结果会失效，需要重新生成。生成后必须在切片器里找到对应独立
对象，确认它被分配给正确的项目耗材，再把项目耗材映射到真实线盘和实际料槽。

:::warning
自由色对象用于提供独立分配入口，不是“看到什么 HEX 就一定能打印出什么颜色”的承诺。最终颜色仍由
真实线材、材料预设、首层、板面和观察光线决定。
:::

## 对象、项目耗材、真实线盘和实际料槽：从左到右逐层核对

### 对象或零件

对象是几何结构。底板、不同材料的网格、描边、挂环、自由色和其他辅助结构都可能成为独立对象。
一个颜色可以拆成多个对象，多个对象也可以使用同一份项目耗材。

因此，对象数量比肉眼看到的颜色多，不一定是错误；判断标准是每个对象最后是否分配到正确材料。

### 项目耗材

项目耗材是 3MF 内部携带的材料定义和编号。对象列表中的数字通常表示“这个对象使用项目耗材几”，
不是机器的真实 AMS 槽位号。

同样，项目耗材的显示色主要用于识别。显示为红色并不能证明它已经对应到正确品牌、系列、材质和批次
的真实线盘。

### 真实线盘与实际料槽

真实线盘是你手上准备打印的材料；实际料槽是它此刻位于 AMS、ACE、MMU 或外置料架的物理位置。

正确顺序是：

1. 先确认每个对象应该使用哪份项目耗材；
2. 再确认项目耗材对应哪一卷真实线盘；
3. 最后把这卷线盘映射到它所在的实际料槽；
4. 执行真实切片，并翻到显色面逐层检查。

项目耗材 3 可以映射到实际 AMS 1。编号相同不是目标，真实对应正确才是目标。完整操作步骤请看
[把 Lumina 3MF 正确导入切片器](pathname:///zh/docs/tutorials/3mf-slicer-material-mapping/)。

## 想改变结果时，应该改哪一个数字？

| 你的目标 | 优先调整 | 先不要动 |
| --- | --- | --- |
| 渐变色带太明显，目标色组太少 | 量化颜色数 | 不要先增加真实线盘 |
| 量化很细，但整体仍缺某个色域 | 耗材档案或 LUT、已选耗材 | 不要只继续提高量化数 |
| 想增加 Z 向混色组合 | 叠色层数，并同步评估耗时与厚度 | 不要把层数当作料槽数 |
| 某块区域必须用指定线材 | 标记自由色、重新生成并在切片器分配 | 不要相信屏幕 HEX 已经自动装料 |
| 切片器里对象很多 | 检查对象到项目耗材的分配 | 不要用对象总数推断颜色总数 |
| 项目耗材编号与 AMS 不同 | 做项目耗材到真实料槽的映射 | 不要为追求同号而随意换料 |

## 两个完整例子

### 例 1：48 个量化色、4 卷耗材、5 层叠色

一张插画在高保真模式中设为 48 个量化色，使用白、红、黄、蓝 4 份耗材档案，叠色层数为 5。

这代表：

- 图像被整理成最多约 48 个目标色组；
- 每个目标色从 4 种材料中组合一份最多 5 层的配方；
- 不需要 48 卷料，也不需要 48 个 AMS 槽位；
- 生成后的对象数可能因底板、附件或几何拆分而大于 4。

### 例 2：把一个红色 Logo 标为自由色

预览后，选中 Logo 对应的匹配红色并标记为自由色，然后重新生成。

这代表：

- 原来的量化颜色数没有改变；
- Lumina 为命中区域增加可独立选择的自由色对象；
- 你仍需在切片器里把这个对象指定给准备好的真实红色线盘；
- 如果新增了实际材料，仍要确认机器有可用槽位并重新切片。

## 常见问题 Q&A

### 量化颜色数设为 256，需要 256 个料槽吗？

不需要。它只是高保真模式的目标色组上限。多个目标色会共享同一组真实耗材，通过不同叠层配方表现。

### 选了 8 种耗材，为什么叠色层数还是 5？

耗材数量描述可用材料通道，叠色层数描述单个像素配方的 Z 向长度。5 层可以从 8 种材料中选择并重复，
两者没有必须相等的关系。

### 旧 4 色 / 6 色 / 8 色模式与自由色是一回事吗？

不是。旧模式的数字主要表示 LUT 的材料通道；自由色是把一个已匹配结果色区域拆成独立对象，方便
后续手动分配。

### 页面里只选了 4 种耗材，切片器为什么有更多对象？

对象是几何结构。底板、附件、同色的分离区域和自由色都可能增加对象数量，但它们可以继续共用同一份
项目耗材。

### 项目耗材 2 必须装在 AMS 2 吗？

不必须。项目编号和物理槽位属于不同层级。发送打印前，让切片器把项目耗材 2 映射到真实线盘所在的
槽位即可。

### 切片器色块与线盘颜色不一样，会不会打印错？

色块可能只是近似显示。先确认对象、项目耗材、真实线盘和槽位四层关系；如果映射正确，单纯的显示色
偏差不会替换实际线材。映射错误则必须先修正。
