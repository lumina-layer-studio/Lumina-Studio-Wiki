# 管理耗材档案：编辑、复验、历史、导入和导出

完成梯度卡提取后，Lumina Studio 会把结果保存为一份**耗材档案**。它不只是一个名称和
颜色方块，还包含这卷耗材在指定层高下的拟合参数、验证历史和供图像转换使用的数据。

本篇讲清楚档案保存以后最常用的管理操作：怎样找到正确耗材、哪些信息可以直接编辑、
什么时候需要再次验证、怎样查看历史，以及如何用 ZIP 迁移和备份。

:::info 本篇适合谁
已经至少创建或导入一份耗材档案，并能在“耗材管理 → 耗材库”中看到它的用户。第一次
创建档案时，请先阅读[生成并打印梯度卡](pathname:///zh/docs/使用教程2/生成并打印梯度色卡/)
和[拍摄并提取梯度卡](pathname:///zh/docs/使用教程2/拍摄并提取梯度卡/)。
:::

## 第一步：打开耗材库并找到正确档案

1. 打开 Lumina Studio；
2. 点击页面顶部的“耗材管理”；
3. 在第二排标签中选择“耗材库”；
4. 展开对应品牌，再找到需要管理的耗材卡片。

耗材库默认按品牌折叠。品牌名称后的数字表示该组当前包含多少份档案。页面顶部左侧显示
“已选择 / 总数”，右侧提供全选、导入、导出和刷新功能。

<img
  src={require('./assets/01-library-overview.webp').default}
  alt="Lumina Studio 耗材库顶部显示选择数量，并提供全选、导入 ZIP、导出和刷新按钮"
/>

如果刚保存或刚导入的档案没有出现，先点击“刷新耗材库”，再展开对应品牌。不要因为
列表暂时没有更新就重复导入同一个 ZIP。

## 第二步：先读懂耗材卡片

一张卡片代表一份可复用耗材档案。开始编辑或删除前，至少核对名称、品牌、目录名、
验证次数和层高。

<img
  src={require('./assets/02-material-card.webp').default}
  alt="一份 Lumina Studio 耗材档案卡片显示名称、颜色、目录、拟合误差、验证次数、使用次数和层高"
/>

| 字段 | 表示什么 | 使用时注意什么 |
| --- | --- | --- |
| 名称与品牌 | 供用户查找和分组的显示信息 | 同名耗材可能来自不同批次、不同设备或不同验证条件，不能只看名称 |
| 显示颜色与十六进制值 | 页面中用于识别档案的近似颜色 | 不是显示器、切片器和打印实物完全一致的保证 |
| 目录名 | 软件用来区分档案的内部身份 | 编辑显示名称后，目录身份不会随之改名 |
| 创建时间 / 最近更新 | 档案首次保存和最近修改的时间 | 多份近似档案并存时，可用来判断版本先后 |
| 拟合误差 | 当前验证结果与采样数据的平均差异 | 通常在相同条件下越低越接近，但不能跨设备、光线和不同拍摄流程机械比较 |
| 验证次数 | 这份档案累计保存了多少次验证 | 次数多不自动等于更准确，仍要检查每次拍摄质量 |
| 使用次数 | 档案被转换流程调用的记录 | 只说明使用情况，不代表质量排名 |
| 层高 | 生成和提取这份档案时使用的层高 | 后续工作流应按档案层高使用，不要把 0.08 mm 档案当成其他层高的测量结果 |

页面可能出现名称和品牌相同的重复提示。这通常表示库中确实存在多份相近档案。此时应展开
卡片核对目录名、更新时间、验证次数和层高，不要随便删除其中一份。

## 第三步：编辑名称、品牌和显示颜色

点击卡片下方的“编辑信息”，可以修改：

- 耗材名称；
- 品牌；
- 用于界面识别的显示颜色或十六进制色号。

<img
  src={require('./assets/03-edit-dialog.webp').default}
  alt="编辑耗材信息窗口可以修改名称、品牌和显示颜色，同时显示不可直接修改的目录名"
/>

保存后，原目录身份保持不变，已有验证历史也会继续保留。这样可以在不破坏档案引用关系的
情况下修正错别字、统一品牌名称，或把颜色块调整得更容易辨认。

:::warning 编辑显示颜色不等于重新校准
这里修改的是档案的显示信息。它不会重新读取梯度卡照片，也不会重新计算 E、K、C0 等
物理拟合参数。若耗材批次、打印条件或照片已经变化，请使用“再次验证”，不要仅靠改色号
掩盖偏差。
:::

## 第四步：查看每一次验证历史

点击“展开验证历史”，可以按验证序号和时间回看这份档案的历次结果。最新一轮显示在最上方。

<img
  src={require('./assets/04-verification-history.webp').default}
  alt="验证历史窗口按时间显示多次耗材验证的颜色、拟合误差、RMSE、方向和诊断参数"
/>

普通用户优先查看以下内容：

1. **验证时间**：确认这是不是预期批次和拍摄日期；
2. **估算颜色**：观察多次结果是否出现明显跳变；
3. **拟合误差**：在同一套拍摄与提取条件下比较相对变化；
4. **线性 RMSE**：辅助判断模型对采样数据的整体拟合程度；
5. **识别方向**：确认白底、黑底照片没有因方向识别异常而产生可疑结果。

E、K、白底 C0 和黑底 C0 主要是拟合与排障信息。普通使用不需要手工抄写或修改这些值。
当两次结果差别很大时，先排查光照、曝光、白平衡、照片方向、四角选取和色卡是否属于
同一卷耗材，不要只挑误差数字最低的一次就认定为正确。

## 第五步：为现有档案再次验证

出现下列情况时，可以从耗材卡片点击“再次验证”：

- 想用更稳定的光线和相机重新拍摄；
- 更换了同色耗材的新批次，想保留前后记录；
- 怀疑上一轮照片偏色、曝光不一致或角点选择不准确；
- 想增加一轮独立采样，观察结果是否稳定。

进入梯度卡提取页面后，顶部会明确显示“当前正在进行耗材复验”，并列出目标耗材与已累计
验证次数。先确认目标正确，再导入与这次实物梯度卡对应的完整 ZIP，分别上传新的白底和
黑底照片。

<img
  src={require('./assets/05-reverify-target.webp').default}
  alt="再次验证时梯度卡提取页面显示目标耗材、现有验证次数和合并结果入口"
/>

完成提取后选择“合并结果”，本次记录会继续写入同一份档案；不想继续时点击“取消复验”。
复验不会替你判断照片是否可靠，因此仍要按梯度卡教程检查均匀光线、固定曝光、正确方向和
四角位置。

:::tip 不要混用数据
本次导入的 ZIP、白底照片和黑底照片必须对应同一卷耗材、同一次梯度卡生成任务和相同层高。
把不同批次或不同层高的数据混在一次复验里，会让结果失去明确含义。
:::

## 第六步：导出 ZIP 作为备份或迁移文件

1. 勾选需要导出的卡片；
2. 如果要导出全部档案，可以点击“全选耗材”；
3. 点击“导出所选耗材”；
4. 等待浏览器下载一个 Lumina 耗材 ZIP；
5. 把 ZIP 改成容易识别的文件名，并保留到独立的离线位置。

可以一次勾选一份或多份档案。无论选择多少份，软件都会把本次选择打包为一个 ZIP。这个
文件用于 Lumina 耗材档案的备份、跨设备迁移或协作，不是可直接打印的 3MF，也不是普通
图片压缩包。

<img
  src={require('./assets/06-import-export.webp').default}
  alt="耗材库选择一份档案后成功导出 ZIP，顶部同时提供导入耗材 ZIP 的入口"
/>

建议在这些操作前先导出：

- 删除档案；
- 大量修改名称或品牌；
- 更换电脑、重装系统或迁移工作环境；
- 向其他获授权的用户分享自制耗材测量数据。

不要只保留软件当前能读到的本地副本。ZIP 才是更适合迁移和恢复的交换文件。

## 第七步：导入一份或多份耗材 ZIP

1. 点击“导入耗材 ZIP”；
2. 选择一个或多个由 Lumina 导出的耗材归档；
3. 等待导入完成和列表刷新；
4. 检查刚导入档案的名称、品牌、目录、验证次数和层高；
5. 若出现重复提示，展开卡片确认哪一份是原档、哪一份是新导入档。

导入不会直接覆盖现有档案。对于已经安装的同一份软件包修订，系统可能跳过重复版本；
对于其他发生目录重名的归档，系统会另存到带时间后缀的新目录。无论哪种情况，导入后都应
回到耗材库核对结果，而不是只看“导入成功”提示。

以下文件不应当作耗材档案导入：

- 普通图片、3MF、STL、SVG 或项目压缩包；
- 只从耗材 ZIP 内部单独取出的 JSON 文件；
- 手工改坏目录结构、清单或校验信息的压缩包；
- 来源不明、内容和授权范围不清楚的耗材包。

导入失败时，先确认扩展名为 `.zip`，再使用原始导出文件重试。不要反复解压、重命名内部
文件后重新压缩。

## 第八步：确认备份后再删除

“删除耗材”会移除当前本地档案。这个操作不可通过耗材库直接撤销。删除前：

1. 确认目录名和更新时间，避免删错同名档案；
2. 先导出该档案并验证 ZIP 已经下载；
3. 确认当前没有正在依赖它的转换任务；
4. 再阅读确认窗口中的名称、目录和更新时间；
5. 确认无误后才执行删除。

如果只是名称不规范、品牌为空或显示色不好辨认，优先使用“编辑信息”，不需要删除重建。
如果只是测量结果需要更新，优先使用“再次验证”。

## 常见问题 Q&A

### 我改了显示颜色，打印结果会跟着改变吗？

不应把它理解为重新校准。显示色主要用于耗材库里的识别和摘要。真正的材料预测仍依赖档案
中的拟合数据。需要更新测量结果时，应重新拍摄并再次验证。

### 拟合误差越低，这份档案就一定越好吗？

不是。拟合误差适合在相同拍摄、提取和数据条件下辅助比较。错误曝光、偏色、反光或角点
选择不准，也可能让单个数字失去代表性。要同时检查实拍条件、估算颜色和多次历史是否稳定。

### 为什么导入后出现两份同名耗材？

名称和品牌是显示信息，不是唯一身份。导入遇到其他重名目录时会另存而不是直接覆盖，因此
可能产生两份显示名称相同的档案。核对目录名、更新时间、验证次数和层高后，再决定保留哪份。

### 能不能把耗材 ZIP 发给别人？

技术上可以导入，但分享前要确认你对测量数据、品牌资料和包内内容拥有传播权限。收到文件的
用户也应确认耗材型号、批次、层高和验证条件是否适合自己的实际材料。

### 一份 ZIP 里可以有多少份耗材？

取决于你导出时勾选了多少份。单份和批量导出都生成一个 ZIP；导入时也可以一次选择多个
ZIP。档案较多时，建议按品牌或用途分批导出，并在文件名中写清日期和范围。

### 云端耗材和本地耗材是同一回事吗？

不是完全相同。耗材库展示当前安装到本地、可供工作流使用的档案；云端耗材库负责发现和
安装可用的软件包。本文讲的是本地档案的日常管理。带“官方”标记的档案还可能受同步、
更新或回滚规则影响，编辑或删除前更应先导出并确认来源。

### 选档案时还有哪些判断标准？

可继续阅读
[现成耗材档案、自制档案、反推 LUT 与传统 LUT 怎么选？](pathname:///zh/docs/knowledge/material-profile-selection/)。
档案是否存在并不代表一定适合当前线材；品牌、材质、颜色、批次、层高和实际验证条件都要
共同考虑。

### 档案准备好以后，怎样比较不同叠层顺序？

可以进入“配方查询”，按观赏面到基底的顺序组合档案，计算预测颜色，再把多组候选配方生成
验证色块 3MF。完整步骤见
[组合耗材层、预测叠色并生成验证色块](pathname:///zh/docs/knowledge/material-recipe-query/)。

## 本篇验证环境

本文使用当前 Lumina Studio 2 内测版本真实操作验证。测试库正常读取 **182 份**耗材档案；
演示档案包含 **4 次**验证历史；单份档案 ZIP 导出成功。为避免修改真实数据，导入、编辑
名称与品牌、修改显示色和再次导出的往返检查均在独立临时耗材目录完成；原目录身份在编辑后
保持不变，导出包可再次读取。浏览器控制台未出现错误。

软件界面、字段名称和档案格式仍可能随版本调整。实际操作时，以当前版本页面和导入结果为准。
