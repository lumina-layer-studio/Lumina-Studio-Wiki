# 透明镀层怎么设置？为什么切片后看不到？

透明镀层会在作品的显色面外侧增加一层独立结构。它可以保留打印板纹理，也能让作品表面呈现不同于普通显色层的光泽和透光效果。

如果你已经在 Lumina Studio 中启用了透明镀层，切片后却完全看不到它，通常不是文件损坏，而是镀层仍在打印板下方。把整个模型重新放回打印板，再切片一次即可。

透明镀层最终效果会受到透明耗材、打印板纹理、首层状态和观察光线影响。切片软件中的显示颜色只能帮助识别对象，不能准确模拟实物的透明度和反光效果。

## 开始前需要准备什么

- 已经完成图片处理，并能正常预览和生成模型；
- 一卷可以稳定打印的透明耗材，本篇以透明 PETG 为例；
- Bambu Studio、OrcaSlicer，或能够完整打开 Lumina 3MF 的兼容切片软件；
- 与实际打印机、喷嘴和耗材相符的切片配置。

当前版本的透明镀层按 **0.08 mm** 基础层高生成。本篇统一使用 0.08 mm。即使界面中还能看到其他高度选项，也请先按 0.08 mm 完成本教程。

## 第一步：在 Lumina Studio 中启用透明镀层

先完成图片、耗材档案、成品尺寸和基础结构设置，然后打开“高级设置”中的“透明镀层设置”。

1. 开启“启用透明镀层”；
2. 将透明镀层高度设为 **0.08 mm**；
3. 重新预览并生成新的 3MF；
4. 下载刚刚生成的新文件。

<img
  src={require('./assets/01-enable-transparent-coating.webp').default}
  alt="在 Lumina Studio 中启用透明镀层并把高度设为 0.08 毫米"
/>

以前下载的 3MF 不会因为页面设置改变而自动更新。开启镀层后必须重新生成；如果新旧文件名称相同，也要确认自己打开的是刚刚下载的版本。

## 第二步：把 3MF 作为完整项目打开

在切片软件中使用“打开项目”或等效方式载入 Lumina 生成的 3MF，不要把它当作普通 STL 或单个几何模型导入。

打开后，在对象列表中展开模型。正常情况下可以看到一个名为 **Coating** 的对象，它就是透明镀层。

如果对象列表中完全没有 Coating，请回到 Lumina Studio 检查：

- “启用透明镀层”是否仍然开启；
- 开启后是否重新生成过 3MF；
- 当前打开的是否为新下载的文件。

## 第三步：把 Coating 映射到透明 PETG

在切片软件的项目耗材列表中加入你实际准备使用的透明 PETG，并选择与真实线盘相符的材料预设。

然后在对象列表中找到 Coating，把它分配给透明 PETG 对应的项目耗材。其他显色对象仍然保留各自原有的耗材，不要把整个模型一起改成透明材料。

<img
  src={require('./assets/02-map-coating-to-petg.webp').default}
  alt="在 Bambu Studio 对象列表中把 Coating 映射到透明 PETG"
/>

切片软件可能把 Coating 显示为灰色、白色或其他普通色块。这只是界面中的识别色，不代表透明 PETG 打印后的真实效果。请以对象旁的耗材编号、项目耗材名称和实际线盘为准。

## 第四步：让整个模型回到打印板

透明镀层生成在原模型的底面外侧。3MF 刚进入切片软件时，它有时会仍然落在 Z=0 以下，也就是打印板下方。此时对象列表里虽然能看到 Coating，切片软件却会忽略位于打印板以下的部分。

在对象列表中选中**整个模型**，不要只选 Coating，然后点击一次“自动摆放”或“放置到热床”。两种方式都可以，目的只是让模型和镀层一起回到打印板上，同时保留它们原有的相对位置。完成后重新切片。

:::warning
不要单独移动 Coating。单独把它放到热床可能改变镀层与显色层之间的位置关系，造成错位或重叠。如果已经单独移动过，请先撤销，再选中整个模型重新摆放。
:::

## 第五步：从第一层检查切片结果

切片完成后进入“预览”，把观察方向翻到作品的显色面，并把层滑块移动到第一层。依次确认：

1. 第一层能看到覆盖作品有效轮廓的透明 PETG 路径；
2. 显色材料从透明镀层上方开始打印；
3. Coating 没有缺失、错位或超出轮廓；
4. 切片软件不再提示对象位于打印板以下；
5. 打印机、喷嘴和真实料槽仍然对应正确。

只在“准备”页面看到 Coating，并不能证明它已经进入打印任务。只有切片后的第一层出现对应耗材路径，才能确认透明镀层会被实际打印。

## 常见问题

### 对象列表里有 Coating，为什么切片后完全没有？

先选中整个模型，点击一次“自动摆放”或“放置到热床”，然后重新切片。Coating 很可能仍在打印板以下，因此被切片软件忽略。

### 为什么不能只选 Coating 放到热床？

Coating 必须与显色层保持原有位置关系。单独移动它可能造成错位，应让整个模型一起移动。

### 透明镀层高度应该选多少？

当前教程统一使用 0.08 mm。现阶段请不要把界面中的其他数值当作已经准确支持的实际镀层厚度。

### 为什么 Coating 在切片软件里看起来不是透明的？

切片软件显示的是对象识别色，不是透明度模拟。确认 Coating 已经映射到实际使用的透明 PETG 即可。

### 可以使用透明 PLA 吗？

透明镀层不限制固定品牌。本篇使用透明 PETG 作为常见方案；如果改用其他透明材料，请先确认它与作品其他材料、打印温度和层间结合相容，并先打印小样。

## 打印前检查清单

- Lumina Studio 中已经启用透明镀层；
- 透明镀层高度为 0.08 mm；
- 当前打开的是重新生成的新 3MF；
- 对象列表中存在 Coating；
- Coating 已映射到真实使用的透明耗材；
- 已选中整个模型执行“自动摆放”或“放置到热床”；
- 重新切片后，第一层能看到透明耗材路径；
- 其他显色对象和实际料槽没有被误改。

Lumina Studio 与切片软件仍会持续更新，个别按钮名称和位置可能略有变化。最终请以对象列表、真实切片路径和小样打印结果为准。
